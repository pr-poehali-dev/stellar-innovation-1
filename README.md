# Шейдерная Hero-секция

![Preview](https://cdn.poehali.dev/templates/shaders-hero-ru/preview-ru.jpg)

Hero-секция с анимированным градиентным фоном на шейдерах. Использует библиотеку @paper-design/shaders-react для создания плавных mesh-градиентов. Включает хедер с навигацией, основной заголовок с CTA-кнопками и декоративный пульсирующий элемент. Подходит для креативных студий, SaaS-продуктов и лендингов с современным дизайном.
